# Hermes

Hermes e un desktop tool in stile Obsidian per la gestione di vault Markdown con pagine tipizzate e navigazione a grafo.

![Coverage](https://img.shields.io/codecov/c/github/simpego81/Hermes?label=coverage)

## Stack

- Electron
- React + TypeScript
- Vite
- Jest
- ESLint + Prettier

## Quick Start

```bash
npm install
npm run dev
```

## Quality Gates

```bash
npm run lint
npm run format
npm run test
npm run build
```
