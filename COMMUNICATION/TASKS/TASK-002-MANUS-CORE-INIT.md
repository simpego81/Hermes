# TASK-002-MANUS: Inizializzazione Core Hermes

## Obiettivo
Creare il boilerplate dell'applicazione e implementare il parser core per i metadati.

## Requisiti
1. **Project Boilerplate**: Setup di Electron + React + TypeScript.
2. **Markdown Parser**: Implementare un modulo che legga i file .md e ne estragga il Frontmatter (YAML) per identificare:
   - Type (Persona, Task, Objective, etc.)
   - Properties specifiche per ogni categoria.
3. **Graph Engine Base**: Integrare `react-force-graph` e visualizzare i file del vault come nodi semplici (collegati tramite link markdown `[[Link]]`).
4. **UI Base**: Creare una sidebar per i file e un'area principale per il grafo.

## Deliverable
- Codice sorgente nel repo GitHub impostato da Copilot.
- Documento di report (REPORT-002-MANUS-CORE-SETUP.md) nella cartella REPORTS.
