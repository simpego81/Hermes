# TASK-007-MANUS: Aggregazione su Timeline (Deadline Layout)

## Obiettivo
Implementare la visualizzazione temporale dei nodi che possiedono una deadline (Task e Objective).

## Requisiti
1. **Timeline Box Logic**: Creare un'area di aggregazione superiore dedicata alla timeline.
2. **Ordinamento Temporale**: I nodi con deadline devono essere disposti lungo un asse orizzontale (da sinistra a destra) in base alla data di scadenza.
3. **UI Toggle**: Implementare un pulsante nella Toolbar "Timeline View".
4. **Data Handling**: I nodi senza deadline rimangono nel grafo "libero" sottostante o in una zona "No Deadline".

## Deliverable
- Algoritmo di layout per timeline.
- Pulsante Toolbar "Timeline View".
- REPORT-007-MANUS-TIMELINE.md
