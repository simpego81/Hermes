# TASK-001-COPILOT: Progetto Setup & Infrastruttura

## Obiettivo
Impostare l'ambiente di sviluppo condiviso e la pipeline di qualità su GitHub.

## Requisiti
1. **GitHub Repository**: Creare un repo per il progetto "Hermes".
2. **CI/CD Pipeline**: Configurare GitHub Actions per:
   - Build automatica (Electron/React).
   - Linting (ESLint/Prettier).
   - Esecuzione Test (Jest).
3. **Testing Framework**: Inizializzare Jest e fornire un test d'esempio funzionante.
4. **Qualità**: Configurare i badge per:
   - Build Status.
   - Code Coverage.
   - Linter Status.

## Deliverable
- Repository GitHub attivo con pipeline funzionante.
- Documento di report (REPORT-001-COPILOT-SETUP.md) nella cartella REPORTS.
